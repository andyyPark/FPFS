Examples

This directory gives some examples on simulating galaxy images with Galsim; and galaxy detection and shape estimation with FPFS.

simulate noise

This code simulates pure noise with the third-order Lanczos correlation function.

simulate stamp-based galaxies

This code supports simulation of parametric galaxies (using Sersic or bulge-disk model fitted to COSMOS galaxies) and galaxies simulated with random knots (for very small galaxies). The galaxies are render into 64x64 postage stamps. Note the galaxies are isolated here.

simulate galaxies in the universe

This code simulates parametric galaxies (Sersic or bulge-disk) randomly and uniformly distributed within a circle. Note the galaxies are blended here. It supports simulations with different number densities.

process with FPFS detector and shear estimator

This code detects galaxies from images and measure the FPFS shapes for shear inference.

estimate average shear

This code and this code use FPFS shapes to estimate average shear for a galaxy sample.
